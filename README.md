ABC Pharmacy Management System

Technology:

Backend:
- ASP.NET Core Web API
- Clean Architecture
- SOLID Principles
- Repository Pattern
- Dependency Injection
- JWT Authentication
- Serilog Logging
- Fluent Validation
- xUnit Test

Frontend:
- React
- Vite
- Axios
- Redux Toolkit
- React Router


Run Backend:

Open ABC.Pharmacy.sln

Run API project

Swagger:
https://localhost:7226/swagger


Run Frontend:

cd abc-pharmacy-ui

npm install

npm run dev


Frontend:
http://localhost:5173


Features:

1. User Registration

2. Login using JWT

3. Logout

4. Add Medicine

5. View Medicines

6. Create Sale

7. View Sales
